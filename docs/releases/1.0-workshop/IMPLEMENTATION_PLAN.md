# Plan d’implantation — MAVIK GCOS 1.0 Atelier

## Lot 1 — Audit et consolidation

- inventorier les écrans et modules existants ;
- identifier les doublons de données entre devis, planning et atelier ;
- définir un identifiant unique d’intervention ;
- préserver les données locales existantes ;
- établir la navigation cible.

## Lot 2 — Tableau de bord atelier

- synthèse opérationnelle du jour ;
- statuts normalisés des véhicules ;
- alertes de blocage ;
- raccourcis vers dossiers, planning et contrôles.

## Lot 3 — Planning unifié

- consolider planning général, atelier, équipe et ressources ;
- afficher immobilisation et séchage ;
- conserver les règles lundi–vendredi et les horaires validés ;
- empêcher toute confirmation incompatible avec la capacité ou les ressources.

## Lot 4 — Fiche intervention unique

- regrouper client, véhicule, devis, photos, planning, procédure et documents ;
- afficher une chronologie complète ;
- unifier les validations opérateur et direction ;
- préparer les sorties rapport et facture.

## Lot 5 — Navigation et expérience mobile

- menu principal unique ;
- accès permanent à MAVIK ;
- interface adaptée à l’iPhone sans supprimer les fonctions de direction ;
- réduction des écrans techniques visibles dans l’usage quotidien.

## Lot 6 — Stabilisation

- tests du parcours complet ;
- tests des rôles David, Bénédicte et employé ;
- contrôle des anciennes données ;
- vérification du démarrage, de la version et des mises à jour ;
- documentation utilisateur ;
- préparation de la publication 1.0.0.

## Ordre de travail

Le premier développement fonctionnel porte sur le tableau de bord atelier et la couche de consolidation des statuts. Le planning et la fiche intervention s’appuieront ensuite sur cette même source de données afin d’éviter de nouvelles divergences.
